## TÍNH NĂNG CHI TIẾT

### 1. HỆ THỐNG ĐĂNG KÝ & ĐĂNG NHẬP

**Đăng ký tài khoản:**

- Đăng ký nhanh chỉ với username, email và mật khẩu
- Xác minh email để lấy lại mật khẩu
- Hỗ trợ đăng nhập qua **Facebook** và **Google OAuth**
- Ghi nhớ đăng nhập trên thiết bị

**Quản lý tài khoản:**

- Xem thông tin cá nhân: số dư, tổng nạp, ngày đăng ký, lần online gần nhất
- Cập nhật email, số Zalo liên hệ
- Đổi mật khẩu với xác thực mật khẩu cũ
- Quên mật khẩu: gửi link reset qua email
- Hệ thống xác minh tài khoản (KYC) để đăng bán code

### 2. MUA BÁN MÃ NGUỒN

**Duyệt và mua code:**

- **300+ mã nguồn** theo danh mục: Shop game, Facebook tools, gạch thẻ, check scam, website viết báo...
- Xem chi tiết: ảnh demo, mô tả, giá bán, lượt xem
- **Xem demo trực tiếp** trước khi mua
- Thêm vào **giỏ hàng** để mua nhiều code cùng lúc
- Áp dụng **mã giảm giá** khi thanh toán
- Tải file code tự động sau khi thanh toán thành công
- Lịch sử mua hàng chi tiết với mã giao dịch

**Hệ thống đối tác (Partner):**

- Đăng bán code riêng sau khi xác minh tài khoản
- Upload code lên Google Drive, hệ thống tự động phân phối
- Tự đặt giá bán, hệ thống trích hoa hồng
- Theo dõi doanh số bán hàng realtime
- Rút tiền về ngân hàng khi đủ điều kiện
- Lịch sử biến động số dư rõ ràng

### 3. TẠO WEBSITE TỰ ĐỘNG

**Khởi tạo website:**

- Chọn từ **195+ mẫu website** theo danh mục: Shop, Blog, Landing Page, Forum...
- Xem demo trực tiếp mỗi mẫu
- Nhập thông tin: tài khoản admin, mật khẩu, tên miền
- Hệ thống tự động **cài đặt hosting**, **deploy code** và **cấu hình domain** trong vài phút
- Nhận thông tin đăng nhập admin ngay sau khi tạo thành công

**Quản lý website:**

- Dashboard quản lý tất cả website đã tạo
- Xem trạng thái hoạt động realtime
- Gia hạn hosting theo tháng
- Theo dõi thời gian hết hạn

### 4. QUẢN LÝ TÊN MIỀN

**Mua tên miền:**

- Tìm kiếm và kiểm tra tính khả dụng của tên miền
- Đa dạng đuôi: **.com, .vn, .net, .info, .xyz...**
- Xem giá theo từng đuôi tên miền
- Chọn thời hạn đăng ký: 1-9 năm
- Cấu hình Nameserver ngay khi đăng ký
- Tự động đăng ký qua **CloudFlare API**

**WHOIS Domain:**

- Tra cứu thông tin tên miền bất kỳ
- Xem chi tiết: người đăng ký, ngày hết hạn, nameserver
- Kiểm tra trạng thái domain

**Quản lý domain:**

- Xem danh sách tên miền đã mua
- Quản lý DNS records
- Gia hạn tên miền tự động
- Lịch sử giao dịch mua miền

### 5. NẠP TIỀN ĐA KÊNH

**Nạp thẻ cào:**

- Hỗ trợ **Viettel, Mobifone, Vinaphone**
- Chọn mệnh giá: 10k - 500k
- Nhập mã thẻ và serial
- **Xử lý tự động** qua TheSieuRe API
- Nhận đúng mệnh giá hoặc chiết khấu theo chính sách

**Nạp ATM/Internet Banking:**

- Hiển thị QR Code chuyển khoản
- Nội dung chuyển khoản tự động (Mã ND + User ID)
- **Quét giao dịch tự động** qua cronjob
- Tiền vào tài khoản trong vài phút

**Nạp Momo:**

- QR Code Momo chuyển khoản
- **Realtime notification** qua TSR API
- Cộng tiền tức thì sau khi chuyển

**Nạp PayPal:**

- Thanh toán quốc tế qua **PayPal SDK**
- Hỗ trợ thẻ Visa/Mastercard
- Tỷ giá tự động VND/USD

**Khuyến mãi:**

- Thêm % giá trị nạp theo sự kiện
- Giảm giá khi mua code/tạo website
- Mã giảm giá dùng một lần

### 6. CHUYỂN TIỀN NỘI BỘ

- Chuyển tiền cho user khác trong hệ thống
- Tối thiểu 10,000đ, tối đa 100,000,000đ
- Lịch sử chuyển/nhận tiền chi tiết
- Ghi nhận IP thực hiện giao dịch

### 7. API CHO DEVELOPER

**Quản lý API Key:**

- Mỗi user có **token API riêng**
- Đổi token bất cứ lúc nào
- Bật/tắt từng endpoint API

**Các API endpoint:**

- **WHOIS Domain**: Tra cứu thông tin tên miền
- **List Code**: Lấy danh sách mã nguồn
- **List Web**: Lấy danh sách mẫu website
- **Buy Code**: Mua code tự động
- **Buy Domain**: Mua tên miền tự động
- **Create Website**: Tạo website tự động
- **Info Domain**: Thông tin domain đã mua

**Tài liệu API:**

- Postman Collection đầy đủ
- Xác thực qua Bearer Token
- Response JSON chuẩn RESTful

### 8. QUẢN TRỊ HỆ THỐNG (ADMIN)

**Dashboard:**

- Thống kê realtime: thành viên mới, lượt truy cập, doanh thu, lượt download
- Biểu đồ doanh thu theo ngày/tháng
- Lịch sử dòng tiền chi tiết

**Quản lý thành viên:**

- Danh sách user online/offline
- Xem chi tiết: số dư, lịch sử giao dịch, IP đăng nhập
- Cộng/trừ tiền thủ công
- Khóa/mở tài khoản
- Xóa tài khoản spam

**Quản lý sản phẩm:**

- Thêm/sửa/xóa mã nguồn
- Thêm/sửa/xóa mẫu website
- Thêm/sửa/xóa tên miền
- Duyệt code từ partner
- Ẩn/hiện sản phẩm

**Quản lý giao dịch:**

- Xem lịch sử nạp thẻ, ATM, Momo, PayPal
- Xem lịch sử mua code, tạo website, mua miền
- Xem lịch sử chuyển tiền
- Xử lý khiếu nại nạp tiền sai nội dung

**Quản lý partner:**

- Duyệt yêu cầu rút tiền
- Xem lịch sử bán hàng của partner
- Chỉnh sửa % hoa hồng

**Cấu hình hệ thống:**

- Bật/tắt sự kiện khuyến mãi
- Cấu hình % giảm giá
- Cấu hình CloudFlare API
- Cập nhật thông báo website
- Cấu hình banner trang chủ
- Block IP tự động chống spam

**Xác minh hồ sơ:**

- Duyệt hồ sơ KYC của user
- Xem CMND/CCCD upload
- Phê duyệt/từ chối

### 9. CRONJOB TỰ ĐỘNG

- **NapATM.php**: Quét giao dịch ngân hàng tự động
- **NapMM.php**: Quét giao dịch Momo tự động
- **NapTSR.php**: Callback từ TheSieuRe khi nạp thẻ
- **StatusWeb.php**: Check trạng thái website đã tạo
- **UpdateWebsite.php**: Cập nhật version code cho website

### 10. BẢO MẬT

**Phòng chống tấn công:**

- SQL Injection: Sử dụng prepared statements
- XSS: Lọc input với htmlspecialchars
- CSRF: Token cho form quan trọng
- Brute Force: Rate limiting đăng nhập
- DDOS: Auto block IP sau N requests

**Bảo vệ dữ liệu:**

- Mã hóa password với MD5/Bcrypt
- Session management an toàn
- API rate limiting
- File upload validation (size, type)
- Xác thực 2 lớp cho admin
